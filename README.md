## Install

The only dependency is the [GNU readline library](https://cnswww.cns.cwru.edu/php/chet/readline/rltop.html).

```
$ make
```

## Usage

```
$ ./khol
```

## Features

- Splits command based on whitespace
- Redirection
- History
- Recalling history through `!` & ~ `!-`
- Piping (two processes)
